# AI PubKit

AI PubKit is a global one-click AI publishing connector site for creators, agencies, and small teams.

The first release is a static Astro site for `aipubkit.com` with:

- Global connector positioning
- Supported apps directory
- Request-an-app intake page
- AiToEarn deployment guide
- MCP publishing guide
- Deployment service page
- SEO and GEO files: sitemap, robots, llms.txt, llms-full.txt, structured data

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
