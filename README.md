# AI PubKit

AI PubKit is a global one-click AI publishing site for creators, agencies, and small teams.

The first release is a static Astro site for `aipubkit.com` with:

- Global one-click publishing positioning
- Supported apps directory
- Request-an-app intake page
- One-click publishing workflow page
- SEO and GEO files: sitemap, robots, llms.txt, llms-full.txt, structured data

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
